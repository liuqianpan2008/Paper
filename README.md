## 项目搭建
1.下载最新<a href="https://github.com/liuqianpan2008/Paper/releases">构建</a><br>
2.下载bd.sql并导入mysql数据库<br>
3.将构建文件上传到服务器<br>
4.输入 'java -jar springbook.jar' 运行后端<br>
